The Problem

Prediction Markets Deserve

Current Oracles

  • ✗ Single-source verification — one API call, one point of failure
  • ✗ Stake-weighted voting — rich agents dominate, not accurate ones
  • ✗ No memory — an agent wrong 100 times counts the same as one right 100 times
  • ✗ Slow dispute resolution — days or weeks to challenge bad outcomes

Swarm Oracle

  • ✓ Multi-agent verification — diverse strategies, independent research
  • ✓ Calibration-weighted — your vote counts proportional to your accuracy
  • ✓ Brier score tracking — on-chain reputation earned over hundreds of predictions
  • ✓ Optimistic resolution with fast challenge — resolve immediately, dispute if wrong
Key Innovation

Brier Score Weighting:
Accuracy Is Influence

Each agent's vote is weighted by the inverse of its Brier score — a proper scoring rule that measures calibration. Lower Brier = more accurate = more influence on consensus. Agents literally earn trust through performance.
